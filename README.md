# Workflow-CI_RaihanFahrezi

Repository Kriteria 3 untuk proyek **Membangun Sistem Machine Learning**.

Project ini menjalankan retraining model Telco Customer Churn menggunakan MLflow Project dan GitHub Actions.

## Run Lokal

```bash
cd MLProject
python -m mlflow run . --env-manager=local
```

## Run GitHub Actions

Push ke branch `main`, lalu cek tab Actions.
